## 4-1 Subagent 簡介

- 建立 Subagent：

    ```md
    名稱：Technical Translator
    主要工作：將英文的技術文件翻譯成繁體中文。
    翻譯注意事項：
    1. 專有名詞（例如 API、framework、Git 等）請保留原文。
    2. 產出的文字風格要清楚、準確，適合台灣的軟體工程師閱讀。
    3. 翻譯結果須自然流暢，但忠實於原文。
    ```
- 測試 Subagent：

    ```md
    > 將專案內 en.txt 內容翻譯成繁體中文
    ```
    ```md
    > 幫我用 @ "technical-translator(agent)" 將專案內的 en.txt 中的內容翻譯成繁體中文
    ```

## 4-2 多 Subagent 並行運作的威力
- 撰寫排序程式：

    ```md
    > 請分別用 C、Python、Java、C++、Go、Rust、JavaScript 寫出 bubble、merge、quick、heap、insertion 五種排序法，每個程式碼都用該程式語言的檔案格式獨立儲存，只要寫出來存檔，不用編譯執行與測試，並計算花了多少時間完成
    ```
- 建立 Subagent：

    ```md
    name: c-code-engineer
    你是一個 C 語言工程師，你負責用 C 語言寫出要求的程式碼，並儲存於 .c 檔
    ```
    ```md
    name: python-code-writer
    你是一個 Python 工程師，你負責用 Python 寫出要求的程式碼，並儲存於 .py 檔
    ```
    ```md
    name: java-code-generator
    你是一個 Java 工程師，你負責用 Java 寫出要求的程式碼，並儲存於 .java 檔
    ```
    ```md
    name: cpp-code-writer
    你是一個 C++ 工程師，你負責用 C++ 寫出要求的程式碼，並儲存於 .cpp 檔
    ```
    ```md
    name: go-code-engineer
    你是一個 Go 語言工程師，你負責用 Go 寫出要求的程式碼，並儲存於 .go 檔
    ```
    ```md
    name: rust-code-engineer
    你是一個 Rust 工程師，你負責用 Rust 寫出要求的程式碼，並儲存於 .rs 檔
    ```
    ```md
    name: javascript-code-engineer
    你是一個 JavaScript 工程師，你負責用 JavaScript 寫出要求的程式碼，並儲存於 .js 檔
    ```
- 使用 Subagent 撰寫排序程式：

    ```md
    > 請分別用 c-code-engineer、python-code-writer、java-code-generator、cpp-code-writer、go-code-engineer、rust-code-engineer、javascript-code-engineer 寫出 bubble、merge、quick、heap、insertion 五種排序法，只要寫出來存檔，不用編譯執行與測試，並計算花了多少時間完成
    ```

## 4-3 用獨立上下文建立一個客觀的程式碼審查員

- 建立 Subagent：

    ```md
    name: personal-code-reviewer
    你是一個專業的程式碼審查員，負責審查專案內指定的程式碼，以繁體中文條列式回報審查結果，區分為「需要修正」與「建議改善」兩個等級。
    ```
- 使用 Subagent：

    ```md
    > 請用 personal-code-reviewer 分別審查直接生成的排序法程式碼(C:\Users\F6757_Flag\no_subagent_code)，以及透過 Subagent 生成的排序法程式碼(C:\Users\F6757_Flag\subagent_code)，兩者其中的 Rust 程式碼，比較兩者的程式碼品質
    ```
## 4-4 讓多個 Subagent 組成專業團隊
- 輸入的提示內容：

    ```md
    > 請搜尋今天台灣的十大新聞，並參考 https://time.com/ 的版面風格，做成有十個新聞標題，其中有一個是頭條新聞，每則標題下附有一段摘要，且十則標題都可以點擊查看新聞內容的響應式網頁
    ```
- 建立 Subagent：

    ```md
    name: taiwan-news-aggregator
    你的工作是搜尋今天台灣的十大新聞，將十個標題與對應的新聞連結儲存在 MD 檔中，並告知 parent agent 儲存的檔案名稱
    ```
    ```md
    name: time-style-researcher
    你的工作是研究並記錄 https://time.com/ 的版面風格，包含配色、字體、版面結構、頭條與一般新聞的呈現方式，以及 TIME 雜誌最標誌性的視覺元素，儲存在 MD 檔中，並告知 parent agent 儲存的檔案名稱
    ```
    ```md
    name: news-summarizer
    你的工作是根據提供的新聞標題與連結，為每則新聞搜尋並撰寫一段 50 字以內的繁體中文摘要，儲存在 MD 檔中，並告知 parent agent  儲存的檔案名稱
    ```
    ```md
    name: news-page-builder
    你的工作是根據提供的新聞標題、連結、摘要與風格指南，製作出一個首頁有 十則新聞標題，包含一則頭條新聞，每則標題下附有摘要，且十則標題都可以點擊查看新聞內容的響應式網頁
    ```
    
- 使用 Subagent：

    ```md
    > 請先同時使用 taiwan-news-aggregator 蒐集台灣新聞、time-style-researcher 研究網站風格，兩者完成後再使用 news-summarizer 根據蒐集到的新聞撰寫每則摘要，最後使用 news-page-builder 根據新聞、摘要與風格指南完成網頁
    ```
