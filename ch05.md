# 第 5 章 用 MCP 與 Skills 幫 Claude Code 生出超能力

## 5-2 內建 MCP：Claude in Chrome

### 安裝並啟用 Claude in Chrome

- [Claude in Chrome 擴充功能](https://chromewebstore.google.com/)

- 啟動 Claude Code 時同步連接 Chrome：

```md
claude --chrome
```

- 設定每次啟動時自動連接（在對話框中輸入）：

```md
/chrome
```

### 下載「臺灣房價地圖」專案

- [臺灣房價地圖 GitHub](https://github.com/FlagTech/test_buyhouse/tree/update_data)

- 複製專案：

```md
git clone -b update_data https://github.com/FlagTech/test_buyhouse.git
cd test_buyhouse
```

- 啟動應用程式：

```md
uv run wsgi.py
```

### 測試使用 Claude in Chrome

- 實戰範例一：本地網頁除錯

```md!
請幫我開啟 http://127.0.0.1:5000，讀取瀏覽器 Console 的錯誤訊息，找出圖表無法顯示的原因
```

- 實戰範例二：操作需要登入的服務

```md!
我在桌面上有一份 sales_data.csv，請幫我在 Google 表單中新增此份資料，把 CSV 中的資料整理後填入
```

## 5-3 使用 Playwright MCP 進行自動化測試

### Claude Code 的 MCP 安裝和管理

- [Playwright MCP GitHub](https://github.com/microsoft/playwright-mcp)

- 標準安裝：

```md
cd test_buyhouse
claude mcp add playwright npx @playwright/mcp@latest
```

- 確認 Playwright MCP 連線狀態：

```md
claude mcp get playwright
```

- 全域安裝 MCP（在所有專案皆可使用）：

```md
claude mcp add --scope user playwright -- npx @playwright/mcp@latest
```

#### 管理 MCP 工具的常用命令

```md
# 確認特定 MCP 工具的狀態
claude mcp get <name>

# 列出 MCP 工具清單
claude mcp list

# 刪除特定 MCP
claude mcp remove <name>
```

### 用 Playwright MCP 測試網頁應用程式

- 提示內容：

```md!
請透過 playwright 開啟 http://127.0.0.1:5000/，並測試我的網頁應用程式功能是否正常
```

- 提示內容：

```md!
請幫我修正所發現的錯誤
```

- 提示內容：

```md!
幫我依序點擊 SVG 地圖中 「全部」 的縣市，確認每個縣市的資料都正常
```

- 解放 MCP tokens 限制(`~/.claude/settings.json` 的 `env` 區塊)：

```md
{
  "env": {
    "MAX_MCP_OUTPUT_TOKENS": "90000"
  }
}
```

## 5-4 使用 Skills 和 Playwright CLI 打包完整測試流程

### 設定 Playwright CLI 環境

- 全域安裝：

```md
npm install -g @playwright/cli@latest
```

- 下載預設瀏覽器：

```md
playwright-cli install-browser
```

- 安裝官方 Skills 至 Claude Code：

```md
playwright-cli install --skills
```

- 查看所有可用命令：

```md
playwright-cli --help
```

### 用 /playwright-cli 即時測試

- 載入官方 Skill：

```md
/playwright-cli
```

- 提示內容：

```md!
請開啟臺灣房價地圖首頁（http://localhost:5000），對網站功能進行主要測試
```

### 建立自訂 playwright-test Skill

- 提示內容：

```md!
請將剛剛使用 playwright-cli 的完整測試流程建立成 Skill，依序測試臺灣房價地圖網站的核心功能。在專案的 .claude/skills/playwright-test/ 建立一個 SKILL.md，frontmatter 設定：name 為 playwright-test、disable-model-invocation 設為 true、allowed-tools 則加入你所需要的命令。
```

### 使用 /playwright-test 啟動測試流程

```md
/playwright-test
```

- 新增測試項目：

```md!
請在 playwright-test SKILL 中新增測試項目，依序點擊 SVG 地圖中的所有縣市，確認每次點擊後城市數值都會更新
```

- 用 /usage 檢視 MCP 與 Skills 的使用量：

```md
/usage
```
