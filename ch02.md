## web_prd.md

```md
# 網頁版產品需求

## 啟動網頁版

- 原程式執行時若沒有加上任何命令行引數，就啟動成為網頁伺服器，等候 5050 連接埠準備連線，顯示網址
- 若有加上命令行引數，就依照原命令行工具運作

## 前端網頁

放置在 front_end 資料夾中。

## 網頁外觀

- 背景使用高科技感的底圖，自行生成
- 版面如下:

    ```
                    +----------------+
   要加浮水印的 PDF：|                |
                    +----------------+
                    +----------------+
           輸出檔名：|                |
                    +----------------+
                    +----------------+
         浮水印文字：|Confidential    |
                    +----------------+
                    +------+
                    | 開始 |
                    +------+
    ```
## 運作流程

1. 按一下 `要加浮水印的 PDF` 欄位即可選取 PDF 檔上傳
2. 未上傳 PDF 前 `輸出檔名` 欄位無法輸入
3. 上傳 PDF 後，`輸出檔名` 欄位預設為上傳 PDF 檔名稱加上 "_wm" 結尾
4. `浮水印文字` 欄位預設為 "Confidential"
5. 按 `開始` 後處理加上浮水印，完成後在按鈕下方顯示下載連結
6. 上傳的 PDF 檔與處理好的 PDF 檔應該要放在 cache 資料夾下，並且不要提交到儲存庫中
```
